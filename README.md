# Brainly
Brainly is a Fullstack web application which lets you store all of your favorite links in one place which you can then share with anyone
- Users can SignUp and Login
- Users can store any link such as a Youtube video or a Twitter tweet using its url and give it a title
- Users can view all their saved link's content on the dashboard itself
- Users can enable sharing which will generate a unqiue link which they can give to anyone who can visit that link to view the user's saved content

The project is built in TypeScript<br>
Expressjs is used for Backend with JWT authentication and hashed password<br>
Reactjs is used for Frontend with reusable components fully optimized<br>
Mongodb is used for the Database with strict schema and relations
